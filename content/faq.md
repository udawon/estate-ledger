# 자주 묻는 질문 (FAQ)

## Q1. 서비스 이용에 비용이 드나요?

**A:** 현재 서비스는 완전 무료로 제공됩니다. 회원가입 없이 주소만 입력하면 바로 분석 결과를 확인할 수 있습니다.

---

## Q2. 어떤 주소 형식을 사용해야 하나요?

**A:** 도로명 주소를 사용하는 것을 권장합니다. 예: `서울특별시 강남구 테헤란로 123`. 지번 주소도 입력 가능하지만, 도로명 주소가 더 정확한 결과를 제공합니다.

---

## Q3. 분석 결과는 얼마나 정확한가요?

**A:** 현재 MVP 버전은 좌표 기반 목(Mock) 데이터를 활용합니다. 향후 공공데이터포털 API(대중교통 정보, 범죄 통계, 환경 데이터 등)와 연동하여 실제 데이터 기반의 정확한 분석을 제공할 예정입니다.

---

## Q4. 종합 점수는 어떻게 계산되나요?

**A:** 교통(30%) + 상권(25%) + 환경(25%) + 안전(20%)의 가중 평균으로 계산됩니다. 각 카테고리는 0~100점으로 평가되며, 종합 점수 기준으로 A~F 등급이 부여됩니다.

---

## Q5. 전국 모든 지역을 분석할 수 있나요?

**A:** 현재 MVP 버전에서는 전국 모든 주소에 대해 분석 결과를 제공합니다. 단, 실측 데이터가 아닌 좌표 기반 목 데이터를 사용하므로 지역별 편차가 있을 수 있습니다.

---

## Q6. 분석 결과를 저장할 수 있나요?

**A:** 현재는 세션 내에서만 결과가 유지됩니다. 브라우저를 새로고침하거나 닫으면 결과가 사라집니다. 향후 결과 저장 및 비교 기능을 추가할 예정입니다.

---

## Q7. 등급 A와 B의 차이는 무엇인가요?

**A:** A등급은 90점 이상으로 교통·상권·환경·안전 전 항목이 우수한 프리미엄 입지입니다. B등급은 75~89점으로 대부분의 항목이 양호하며 거주와 투자에 적합한 입지입니다.

---

## Q8. 상업용 부동산도 분석할 수 있나요?

**A:** 네, 주거용·상업용·오피스용 부동산 모두 분석 가능합니다. 상업용의 경우 상권(25%) 지표를 특히 참고하시면 유용합니다. 다만 현재 MVP에서는 용도별 가중치 조정 기능은 미지원입니다.

---

## Q9. 분석 결과에 이의가 있으면 어떻게 하나요?

**A:** 현재 MVP 버전은 목 데이터 기반으로 실제 현지 상황과 다를 수 있습니다. 중요한 부동산 결정은 반드시 현장 방문과 전문가 상담을 병행하시기 바랍니다.

---

## Q10. 향후 어떤 기능이 추가될 예정인가요?

**A:** 다음 기능들을 순차적으로 추가할 예정입니다.
- 실시간 공공데이터 API 연동 (교통, 범죄, 환경 통계)
- 분석 결과 저장 및 비교 기능
- 지도 기반 시각화 (카카오맵/네이버맵 연동)
- 주변 유사 입지 비교 분석
- 시계열 데이터 기반 입지 변화 추적
