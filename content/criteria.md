# 점수 산출 기준 상세

## 종합 점수 계산 방식

총 점수는 4개 카테고리의 가중 평균으로 계산됩니다.

```
종합 점수 = (교통 점수 × 0.30) + (상권 점수 × 0.25) + (환경 점수 × 0.25) + (안전 점수 × 0.20)
```

## 등급 기준

| 등급 | 점수 범위 | 의미 |
|------|-----------|------|
| A | 90점 이상 | 최우수 입지 |
| B | 75~89점 | 우수 입지 |
| C | 60~74점 | 보통 입지 |
| D | 40~59점 | 미흡한 입지 |
| F | 39점 이하 | 불량 입지 |

## 카테고리별 평가 기준

### 교통 접근성 (가중치 30%)
- 지하철역까지의 도보 시간 (200m 이내 최고점)
- 버스 정류장 수 (반경 300m 기준)
- 광역버스 및 환승 편의성

### 상권 분석 (가중치 25%)
- 편의점·마트 수 (반경 500m 기준)
- 음식점 밀도 (반경 300m 기준)
- 백화점·쇼핑몰 접근성

### 환경 지수 (가중치 25%)
- 공원 면적 및 접근성 (반경 1km 기준)
- 학교 (초·중·고) 접근 거리
- 소음 지수 (도로 유형·교통량 기반)

### 안전 지수 (가중치 20%)
- 최근 5년 범죄 발생 지수 (지역별 통계)
- 소방서 도달 시간 (반경 내 소방서 수)
- CCTV 설치 밀도

## 점수 신뢰도

현재 MVP 버전은 좌표 기반 목(Mock) 데이터를 사용합니다. 향후 정식 서비스에서는 공공데이터포털 API와 연동하여 실측 데이터를 기반으로 점수를 산출할 예정입니다.
