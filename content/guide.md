# 입지분석 사용 가이드

## 시작하기

estate-ledger는 부동산 입지를 AI로 자동 분석해주는 서비스입니다. 주소 하나만 입력하면 교통·상권·환경·안전 4가지 지표를 종합한 점수를 30초 안에 받을 수 있습니다.

## 분석 방법

1. **주소 입력**: 분석하고 싶은 부동산의 도로명 주소를 입력합니다.
   예: `서울특별시 강남구 테헤란로 123`
2. **분석 시작**: "분석하기" 버튼을 클릭합니다.
3. **결과 확인**: 종합 점수와 카테고리별 점수를 확인합니다.
4. **세부 내용 검토**: 각 카테고리의 세부 항목을 참고해 입지를 판단합니다.

## 주의사항

- 도로명 주소를 사용하면 더 정확한 결과가 제공됩니다.
- 현재 서비스는 MVP 버전으로 목(Mock) 데이터 기반으로 제공됩니다.
- 실제 투자 결정 시에는 전문가 상담을 병행하시기 바랍니다.
