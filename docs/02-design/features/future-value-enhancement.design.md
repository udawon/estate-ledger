## 4. extractDistrict 개선 (district-data.ts)

현재 정규식: `/([가-힣]+구)/` — 구(區)만 매칭

개선 불필요: Kakao API 설정 시 `getRegionCode()` → `sigungu`로 보완되므로,
시(市) 단위 extractDistrict 수정은 YAGNI. 단, Kakao 미설정 시 market address에서
"시" 패턴도 fallback으로 추가.

```ts
export function extractDistrict(address: string): string {
  // 구(區) 우선 매칭
  const guMatch = address.match(/([가-힣]+구)/);
  if (guMatch) return guMatch[1];
  // 구 없으면 시(市) 매칭 (경기도 독립시: 광명시, 하남시 등)
  const siMatch = address.match(/([가-힣]+시)(?!\s*[가-힣]+구)/);
  return siMatch ? siMatch[1] : '';
}
```
